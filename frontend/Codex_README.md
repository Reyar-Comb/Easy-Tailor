由于我正在学习Golang中，但是不想浪费时间写前端，我需要你辅助我完成前端UI界面的工作，  根据我的需求设计好接口并写好文档交付给我，我来用golang做接下来的工作。  
软件的整体逻辑如下：用户可以导入一个或多个音频文件，然后在操作面板进行简单更改编辑并导出。
请你只在frontend里创建/修改vue文件，根据我的阶段性指示完成阶段性工作并等待我的进一步请求。  

## 第一阶段
请你根据我设计好的UI创建好前端界面，并写好一个md文档告诉我你的具体逻辑与接口。
